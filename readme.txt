QUADRATUM
=========

http://code.google.com/p/quadratum

BUILDING
========

Run ./util/build.sh

RUNNING
=======

Run ./util/run.sh

BUILDING A JAR
==============

Run ./util/jar.sh
The bin, img, and maps folders must be distributed along with the .jar.

RUNNING TESTS
=============

Run ./util/test.sh

RUNNING THE MAP EDITOR
======================

Run ./util/map.sh
